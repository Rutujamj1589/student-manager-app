# Student Manager App

## Description

A Python-based Student Management System that helps manage student records efficiently. The application stores data permanently using JSON file handling.

## Features

* Add Student
* View Students
* Delete Student
* Check Student Result
* Update Student Information
* Persistent Data Storage using JSON
* Duplicate Roll Number Validation

## Technologies Used

* Python
* JSON File Handling

## Concepts Used

* Dictionaries
* Loops
* Conditional Statements
* CRUD Operations
* File Handling
* Data Validation

## How to Run
1. Make sure Python is installed on your system.
2. Download or clone the project files.
3. Open the project folder in VS Code.
4. Open the terminal in VS Code.
5. Run the following command:

```bash
student_manager_app.py
```

6. The Student Manager App menu will appear on the screen.
7. Select an option and start managing student records.

8. ### Note

* Student data is automatically saved in `students.json`.
* The data remains available even after closing and reopening the program.
